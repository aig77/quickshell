# quickshell config

Vibe coded Hyprland shell using [Quickshell](https://quickshell.outfoxtrot.net/). Built incrementally with Claude.

## Stack

- **Quickshell** — shell framework (QML-based)
- **Hyprland** — Wayland compositor
- **Stylix / Home Manager** — color theme generation
- **Catppuccin Mocha** — fallback color palette

## Components

| Component | Description |
|-----------|-------------|
| `Bar` | Top status bar |
| `Dock` | macOS-style taskbar — auto-hides, shows open windows |
| `Launcher` | Spotlight-style app launcher |
| `NotificationStack` | Notification + MPRIS now-playing overlay |
| `Overview` | Workspace overview grid |

## Theme

Colors are loaded from `~/.cache/stylix/colors.json` at runtime (generated by Home Manager / Stylix). Falls back to Catppuccin Mocha if the file is absent.

The Overview uses its own color pipeline via `components/overview/common/Appearance.colors.qml` (also generated by Home Manager) and is activated by setting `colorSource: "matugen"` in `components/overview/config.json`.

## Structure

```
shell.qml                          # entry point
components/
  Bar.qml
  Colors.qml                       # global color singleton
  Dock.qml
  Launcher.qml
  notifications/
    NotificationStack.qml
    NotificationItem.qml
    MprisNotificationItem.qml
  overview/                        # third-party overview widget
    common/
      Appearance.qml               # color + animation system
      Appearance.colors.qml        # generated by Home Manager
      Config.qml                   # reads overview/config.json
    ...
```
